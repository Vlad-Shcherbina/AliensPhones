Структура каталогов:
    Phones - вот этот самый репозиторий
        Logic
        Mobile
    Cards - содержимое архива Cards.zip


Инструменты:
    python 2.7.*
    eclipse с плагином mtj
    git
    dropbox
    firefox (для просмотра и печати карточек)

Папку Phones можно считать эклипсовым воркспейсом с двумя проектами:
    Logic - движок, контент, консольный интерфейс; по совместительству там же лежат скрипты для генерации и подсасывания кодов
    Mobile - мобильный интерфейс только

В папке Cards, разумеется, карточки и генератор их. Используемые коды в Cards/codes.

Коды и карточки, генеримые скриптами, были заморожены в момент отправки на печать.
Для предотвращения случайного запуска соответствующие скрипты были лишены своего исконного расширения.
